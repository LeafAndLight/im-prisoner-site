# I'm Prisoner — GitHub Pages site

This folder contains a simple landing page and a privacy policy for **I'm Prisoner**.

## Files
- `index.html` — landing page
- `privacy.html` — privacy policy
- `assets/` — screenshots

## How to publish on GitHub Pages
1. Create a new GitHub repository (e.g., `im-prisoner-site`)
2. Upload all files from this folder to the repo root
3. Go to **Settings → Pages**
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)`
5. Save — your site will be available at the URL GitHub shows.

## Update screenshots
Replace files in `assets/` and keep the same names (or update the `<img src="...">` paths in `index.html`).
